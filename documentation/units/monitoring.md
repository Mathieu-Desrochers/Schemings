with-monitoring
---------------
Invokes a procedure with monitoring.

__ip-address__  
A statsd server ip address.

__port-number__  
A statsd server port number.

__procedure__  
A procedure invoked with a monitor.

monitoring-signal
-----------------
Signals an event.

__monitor__  
A monitor.

__event-name__  
An event name.

monitoring-set-value
--------------------
Sets a monitored value.

__monitor__  
A monitor.

__name__  
A value name.

__value__  
A value.

monitoring-timing
-----------------
Reports a timed operation.

__monitor__  
A monitor.

__operation-name__  
An operation name.

__duration__  
A duration.
